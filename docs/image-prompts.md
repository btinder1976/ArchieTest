# Image Generation Prompts for Archie My Kuya

Use these prompts to create placeholder images that match the website theme. Save generated images in `assets/images/` and update `index.html` or CSS when ready.

## 1. Hero Image

Create a cinematic science-themed hero image for a website called "Archie My Kuya." Show a friendly Filipino biology teacher in a futuristic classroom-laboratory, surrounded by glowing DNA strands, microscope slides, plants, molecules, and a soft AI hologram interface. Include subtle welding sparks in the background as a secondary theme. The style should be professional, inspiring, vibrant, educational, and high-tech. Wide website hero composition, dramatic lighting, no text, no logos.

## 2. Biology Lesson Card Image

Create a colorful 3D educational illustration of a living cell as a tiny city. Show the nucleus as a command center, mitochondria as energy stations, membranes as borders, and organelles as friendly labeled-looking structures without readable text. Bright science classroom style, clean, modern, kid-friendly but professional.

## 3. Ecology Lesson Image

Create a vivid ecosystem science illustration showing plants, insects, birds, water, sunlight, soil layers, and energy flow as glowing lines. Make it look like a dynamic classroom poster with depth and motion. No text.

## 4. AI Learning Image

Create a positive educational AI image: a teacher using a transparent holographic assistant to create science quizzes, diagrams, and study notes. Include DNA, molecules, and notebook elements. Make the AI look like a helpful learning tool, not a robot takeover. No text.

## 5. Welding Science Image

Create a safe welding science image showing metal, heat, sparks, protective gear, and molecular structure overlays. Blend workshop craftsmanship with science education. Dramatic but clean, no readable text.

## 6. Virtual Lab Background

Create a futuristic virtual biology lab background with microscopes, glass slides, plants, glowing DNA, transparent screens, beakers, and classroom learning energy. Wide banner format, no text.

## Suggested style words

- cinematic science education
- futuristic biology classroom
- glowing molecular overlays
- friendly teacher mentor
- professional STEM website
- vibrant but trustworthy
- high contrast dark lab background
- colorful DNA and microscope imagery
